# Contributing to Prosthetic Outreach Initiative

Thank you for your interest in contributing to our prosthetic outreach initiative!

## How to Contribute

### 1. Claim a Task
- Browse the [open issues](https://github.com/zhub9006/prosthetic-volunteer-coord-2025/issues) in this repository
- Comment on an issue to claim it (add your name and planned approach)
- Choose tasks from the categories: Outreach, Logistics, Research, Training, Fundraising, Analytics

### 2. Work on Your Task
- Fork the repository if needed
- Create a branch for your work: `git checkout -b task/your-task-name`
- Make your changes and commit with clear, descriptive messages

### 3. Submit Your Work
- Open a Pull Request against the `main` branch
- Link your PR to the relevant issue number
- Include a summary of your changes

### 4. Review & Merge
- Team leads will review your PR
- Once approved, it will be merged into `main`

## Sprint Schedule

We operate on 2-week sprint cycles:
- **Sprint 1**: Outreach database + affordable device catalog
- **Sprint 2**: Supply chain mapping + grant identification
- **Sprint 3**: Dashboard/analytics + training materials

## Communication

- Check issues weekly for updates
- Tag relevant team leads in conversations
- Update issue status as you progress

## Questions?

Open an issue with the "question" label and we'll help you get started!
